Questions fréquentes

Comment créer une publication ?
Cliquez sur le bouton + en bas de l'écran. Rédigez votre message, ajoutez une image ou un document, choisissez la visibilité (Public, Groupe, Emploi) puis cliquez sur "Publier". Votre publication apparaîtra immédiatement dans le fil d'actualité.
Comment changer le thème (clair/sombre) ?
Allez dans "Paramètres" > "Préférences" > "Thème". Choisissez entre le mode Sombre et le mode Clair. Votre préférence est sauvegardée et s'appliquera à toutes les pages.
Comment rechercher des personnes ou du contenu ?
Utilisez la barre de recherche en haut de l'écran. Tapez un nom, un mot-clé ou un sujet. Vous pouvez filtrer les résultats par "Personnes", "Publications" ou "Groupes".
Comment signaler un contenu inapproprié ?
Sur chaque publication, cliquez sur les trois points puis sélectionnez "Signaler". Notre équipe de modération examinera le signalement sous 24h.
Comment gérer mes notifications ?
Allez dans "Notifications". Vous pouvez filtrer par type, marquer comme lu, ou supprimer. Dans les paramètres, vous pouvez choisir quelles notifications recevoir.
Comment supprimer mon compte ?
Allez dans "Paramètres" > "Zone de danger" > "Supprimer le compte". Attention, cette action est irréversible et toutes vos données seront effacées.
Comment modifier mon mot de passe ?
Allez dans "Paramètres" > "Compte" > "Sécurité". Vous pourrez y changer votre mot de passe et activer la double authentification.
