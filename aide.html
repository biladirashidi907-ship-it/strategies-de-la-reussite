Comment créer un compte étape par étape
Créer un compte sur ESTRA LINK est gratuit et prend moins de 2 minutes. Suivez ces étapes simples :
1

Accédez à la page d'inscription

Ouvrez l'application ESTRA LINK. Sur la page de connexion, cliquez sur l'onglet "Inscription" en haut du formulaire.

2

Remplissez votre nom complet

Entrez votre vrai nom (prénom et nom). Minimum 3 caractères. Cela permettra à vos amis de vous reconnaître facilement.

3

Choisissez un nom d'utilisateur

Créez un pseudo unique (minimum 3 caractères). C'est votre identité publique sur ESTRA LINK. Exemple : JeanDu93

4

Entrez votre adresse email

Utilisez une adresse email valide (ex: votrenom@gmail.com). C'est important pour récupérer votre mot de passe si vous l'oubliez.

5

Créez un mot de passe sécurisé

Choisissez un mot de passe d'au moins 6 caractères. Pour plus de sécurité, mélangez lettres, chiffres et symboles. La barre de force vous indique si votre mot de passe est fort.

Astuce : Utilisez une phrase facile à retenir comme "MonChienNoir2026!"
6

Confirmez votre mot de passe

Retapez exactement le même mot de passe dans le champ "Confirmation". Les deux doivent être identiques.

7

Acceptez les conditions

Cochez la case "J'accepte les conditions d'utilisation". Cela confirme que vous respectez les règles de la plateforme.

8

Cliquez sur "Créer mon compte"

Appuyez sur le bouton bleu "Créer mon compte". Si tout est correct, vous serez automatiquement connecté ! 🎉

Félicitations ! Votre compte est créé. Vous pouvez maintenant explorer ESTRA LINK, publier, discuter et rejoindre des groupes.

Questions fréquentes (FAQ)
🔐 Que faire si j'ai oublié mon mot de passe ?

Sur la page de connexion, cliquez sur "Mot de passe oublié ?". Entrez votre adresse email, recevez un code de vérification, puis créez un nouveau mot de passe.

📧 Je n'ai pas reçu le code de vérification

Vérifiez d'abord vos spams ou courriers indésirables. Assurez-vous d'avoir entré la bonne adresse email. Cliquez sur "Renvoyer le code" pour recevoir un nouveau code.

👤 Puis-je changer mon nom d'utilisateur plus tard ?

Oui ! Allez dans Paramètres depuis le menu latéral, puis modifiez votre profil. Le changement de pseudo est possible une fois par mois.

🛡️ Mes données sont-elles en sécurité ?

Absolument. Toutes les données sont cryptées et protégées. Nous ne partageons jamais vos informations personnelles avec des tiers.

🌐 Puis-je utiliser ESTRA LINK sans connexion internet ?

Certaines fonctionnalités de base sont disponibles en mode hors-ligne grâce à la PWA. Vous pouvez consulter vos messages et posts déjà chargés.

📱 L'application est-elle disponible sur mobile ?

Oui ! ESTRA LINK est une PWA (Progressive Web App). Vous pouvez l'installer sur votre écran d'accueil depuis Chrome ou Safari en cliquant sur "Installer l'application".

❌ J'ai un message d'erreur "Email déjà utilisé"

Cela signifie qu'un compte existe déjà avec cette adresse email. Essayez de vous connecter avec cet email, ou utilisez l'option "Mot de passe oublié" pour réinitialiser votre mot de passe.

🔑 Comment créer un mot de passe fort ?

Un bon mot de passe contient : au moins 8 caractères, une majuscule, un chiffre et un symbole. Exemple : ESTRA2026!Link. La barre de force vous guide en temps réel.

Problèmes courants et solutions
Problème : Le bouton "Créer mon compte" ne fonctionne pas

✅ Solution : Vérifiez que tous les champs obligatoires sont remplis et que les mots de passe correspondent. La case des conditions doit être cochée.

Problème : "Nom d'utilisateur déjà pris"

✅ Solution : Choisissez un autre pseudo. Ajoutez des chiffres ou un underscore. Exemple : Jean_2026 au lieu de Jean.

Problème : L'application ne charge pas

✅ Solution : Vérifiez votre connexion internet. Actualisez la page. Si le problème persiste, videz le cache de votre navigateur.
